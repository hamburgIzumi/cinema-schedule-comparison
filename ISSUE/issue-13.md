# [Issue #13] ダミーデータ生成およびダミー警告表示の完全撤去

- **ステータス**: 完了 (Completed)
- **担当**: 役割B (フロントエンド開発)
- **ブランチ**: `feat/issue-13-remove-dummy-data` -> `work`

## 1. 目的
ユーザー指示に基づき、フォールバック用のダミー（デモ）データ（名探偵コナン等の代替サンプルデータ）の自動生成ロジックおよびUI上のダミー警告表示を完全撤去し、実データのみを表示するシンプルな構造にする。

## 2. 実装内容
- `workers/index.js`: `getFallbackMovies`, `getRealCinemaMovies` などのダミーデータ補完コードを完全削除。抽出された実データのみを返却する。
- `js/fetchers/*.js`: `getRealtimeFallbackData` によるダミーデータ補完の停止。
- `js/uiRender.js` & `js/app.js` & `css/components.css`: `demo-data-badge` および「デモデータ表示中」の警告表示ロジック・スタイルを撤去。

## 3. 完了条件
- [x] ダミー（デモ）データが一切生成・挿入されないこと
- [x] 実データのみがマトリクス比較表にレンダリングされること
- [x] 「⚠️ デモ（ダミー）データ」バッジや警告表示が画面から完全に削除されること
