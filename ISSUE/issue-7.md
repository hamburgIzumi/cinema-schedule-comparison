# [Issue #7] 当日を含む7日間の日付選択・切り替え機能の実装

- **ステータス**: 完了 (Completed)
- **担当**: 役割B (フロントエンド開発)
- **ブランチ**: `feat/issue-7-date-select` -> `work`

## 1. 目的
ユーザーが「当日を含む7日間」の任意の日付を選択し、対象日の上映スケジュールおよび空席状況を動的に確認できるようにする。

## 2. 実装内容
- `index.html`: 当日〜6日後までの日付を選択できる日付ナビゲーションタブの追加
- `css/components.css`: 日付タブのアクティブ状態、ホバーアニメーション、レスポンシブスタイルの定義
- `js/fetchers/*.js`: 日付指定パラメータ（`dateStr` / `YYYYMMDD`）を受け取り、指定日のスケジュールを動的取得・生成するロジックへの拡張
- `js/app.js`: 日付選択状態の管理、タブ切り替え時の非同期再取得・描画

## 3. 完了条件
- [x] 画面上に「当日を含む7日間」の日付タブが表示されていること
- [x] 日付タブをクリックすると、選択した日付の上映スケジュールが動的に切り替わり表示されること
- [x] 映画館の動的フェッチに日付パラメータが正常に連動すること
