# [Issue #5] スタイリッシュなダークテーマ UI/UX とレスポンシブ比較表・モーダルの実装

- **ステータス**: 完了 (Completed)

## 1. 目的
ユーザーが直感的に全映画館のスケジュールと空席状況を比較できる、モダンで高級感のあるダークモードWeb画面を構築する。

## 2. 実装内容
- `css/main.css`: ガラスモルフィズム、グラデーションテーマ、レスポンシブCSS変数の作成
- `css/components.css`: タイトル固定列を持つレスポンシブマトリクス表、空席状況バッジ（◎, ◯, △, ×）、ローディングスピナー、モーダルのスタイル
- `js/uiRender.js`: 統合データのHTMLレンダリング、モーダルダイアログ表示制御
- `js/app.js`: 初期化、非同期通信制御、手動更新ボタンイベント
- `index.html`: アプリのメイン画面マークアップ

- [x] ダークモードテーマでレスポンシブ表示に対応していること
